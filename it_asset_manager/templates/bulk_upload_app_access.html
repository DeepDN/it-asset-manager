How to Use
1
Download Template

Download the sample CSV file to see the required format and example data.

2
Fill Your Data

Replace sample data with your actual access information. Keep the headers unchanged.

3
Upload File

Upload your completed CSV file. The system will validate and import your access records.

Tips:
  • Required fields: user_name, application_name, access_level
  • Use YYYY-MM-DD format for dates
  • Existing access will be updated
  • Access levels: admin, write, read
  • Status: active, revoked
